# TechFest 2026 🚀

A modern, responsive web portal designed for a university-level technical festival. This project emphasizes a clean, **Apple-inspired aesthetic** using system typography, generous whitespace, and a high-performance grid layout.

## 🌐 Live Demo
**View the live project here:** [https://priyakantji301.github.io/techfest-2026/](https://priyakantji301.github.io/techfest-2026/)

## ✨ Key Features
*   **Modern UI/UX**: Implements an 18px border-radius and soft box-shadows for a premium card-based design.

*   **Dual-Column Rules**: A side-by-side Flexbox layout separating General Guidelines from Registration Processes.
*   **Interactive Gallery**: A CSS Grid-powered media section with smooth hover-zoom animations for images and videos.
*   **Registration Portal**: A stylized form including fields for Enrollment Number and Password, optimized for user input.
